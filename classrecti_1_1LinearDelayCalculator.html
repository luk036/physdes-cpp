recti::LinearDelayCalculator class

Implements a simple linear delay model.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

In this model, delay is directly proportional to the wire length. Delay = delay_per_unit * length. Capacitance = capacitance_per_unit * length. This model is simpler but less accurate than the Elmore delay model.

std::pair<int, double> recti::LinearDelayCalculator::calculate_tapping_point(TreeNode& node_left, TreeNode& node_right, int distance) override

Determines the tapping point for prescribed-skew (not necessarily zero) using the linear delay model.

Parameters
node_left The left child node.
node_right The right child node.
distance The Manhattan distance between the child segments.
Returns A pair containing the length of the wire extension from the left child and the resulting delay at the tapping point.

This implementation calculates the extend_left such that the delays from the tapping point to the sinks of node_left and node_right are balanced.
