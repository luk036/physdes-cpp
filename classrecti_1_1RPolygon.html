template<typename T>
recti::RPolygon class

Rectilinear Polygon.

Template parameters
T

Contents

RPolygon is a class template that represents a rectilinear polygon. It stores the origin point and a vector of edges that define the polygon. The template parameter T specifies the type of the coordinates of the points.

template<typename T>
recti::RPolygon<T>::RPolygon(std::span<const Point<T>> pointset) explicit constexpr

Constructs a new RPolygon object from a set of points.

Parameters
pointset in A span of Point<T> objects representing the vertices of the rectilinear polygon.

This constructor takes a std::span of Point<T> objects representing the vertices of the rectilinear polygon. The first point in the span is used as the origin of the polygon, and the remaining points are used to construct the vectors that define the edges of the polygon.
